# AP Framework Mod (With Custom UE4SS and PalSchema)

Core APFramework mod for Palworld. This version includes a custom UE4SS configuration and the PalSchema mod.

## Requirements

- UE4SS (Palworld fork) — see [Installation Guide](./APFrameworkMod/ue4ss_setup.md)

## UE4SS Setup

This mod includes a `ue4ss.json` with recommended UE4SS sources. APF Manager will use this to guide installation.
